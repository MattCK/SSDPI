Instantiates class and creates connection to My\-S\-Q\-L database.

Creates connection to My\-S\-Q\-L database based on the arguments passed.

On success, the connection occurred without error and the object is instantiated. On failure, die() is called and the My\-S\-Q\-L error outputted.
