__construct()

__construct(string $host, string $username, string $password, string $databaseName)

Instantiates class and creates connection to MySQL database.

Creates connection to MySQL database based on the arguments passed.

On success, the connection occurred without error and the object is instantiated. On failure, die() is called and the MySQL error outputted.

Parameters

string $host

The name or IP of the host to connect to
string $username

The username of the MySQL user to use
string $password

The password of the MySQL user to use
string $databaseName

The name of the database to connect to
